# BCYF App
Custom mobile application designed for the Berrien County Youth Fair (BCYF) in held annually in Berrien County, Michigan.  The app includes the following functionalities:

- Schedule information for all exhibitor classes, demonstrations, shows, and other events happening at the fair.  Events can be searched by title, class number, division, date, and location.
- A dynamic list of events happening now or coming up soon.
- A "My Schedule" section populated by events starred by the user.
- More functionality coming soon including a map and location information.
